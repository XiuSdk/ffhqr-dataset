Flickr-Faces-HQ-Retouching (FFHQR) is a high-quality image dataset of
retouched human faces. The dataset is released as part of the following paper:

    AutoRetouch: Automatic Professional Face Retouching
    Alireza Shafaei (skylabtech.ai), James J. Little (UBC), Mark Schmidt (UBC)
    https://github.com/skylab-tech/ffhqr-dataset

FFHQR is derived from FFHQ and follows the same license. For the license of 
the original images please see the official FFHQ Github page at

https://github.com/NVlabs/ffhq-dataset
